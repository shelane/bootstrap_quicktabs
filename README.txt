Quicktabs

Description:
-------------------------------------------------------------------------------

This module provides a "renderer" for to the Quicktabs module that enables site-builders to add Bootstrap-style tabs or pills to a page. This module should work with the Bootstrap theme. Some functionality of the Bootstrap tabs were added with Bootstrap 3 and some were removed. The form will specify which position choices require which version.

Installation and Use:
-------------------------------------------------------------------------------
1. Enable module in module list located at administer > structure > modules.
2. Go to admin/structure/quicktabs and click on "Add Quicktabs Instance".
3. Select "bootstrap_tabs" as renderer.
4. The extras folder contains css files for providing Bootstrap 3 styles for Bootstrap 2 users and Bootstrap 2 styles for Bootstrap 3 users. You will have to add this to your Bootstrap sub-theme css files.

Options:
-------------------------------------------------------------------------------
Information on Bootstrap Tabs can be found on the Bootstrap documentation site:
http://getbootstrap.com/components/#nav-tabs

Style:
Tabs (default)
Pills

Position of tabs:
Tabs on the top (default)
Tabs on the left
Tabs on the right
Tabs on the bottom
Tabs justified on the top (Bootstrap 3 only)
Tabs stacked

Effect of tabs
Fade (default off)
